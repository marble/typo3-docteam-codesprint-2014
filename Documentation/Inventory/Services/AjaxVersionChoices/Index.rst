Generate html for VERSION switcher
==================================

::

   Web address:
      http://docs.typo3.org/php/versionchoices.php

   Installed at:
      BASE=/home/mbless/public_html/php

   which is a symlink to:
      /home/mbless/HTDOCS/github.com/marble/typo3-docs-typo3-org-resources.git/webroot/php

   Cannot be viewed via the following line because of an .htaccess file:
      http://docs.typo3.org/~mbless/HTDOCS/github.com/marble/typo3-docs-typo3-org-resources.git/webroot/php/

   Script
      SCRIPT=$BASE/versionchoices.php

   Example:
      http://docs.typo3.org/php/versionchoices.php?url=http://docs.typo3.org/typo3cms/TyposcriptReference/

..

Example:

- `Find available versions of the TSREF <http://docs.typo3.org/php/versionchoices.php?url=http://docs.typo3.org/typo3cms/TyposcriptReference/>`__
- `Find available versions of EXT:sphinx <http://docs.typo3.org/php/versionchoices.php?url=http://docs.typo3.org/typo3cms/extensions/sphinx/>`__
